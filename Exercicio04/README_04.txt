A atividade consiste em acrescentar as instruções no Processador no ciclo de BUSCA e DECODIFICAÇÃO.

No ciclo de Busca, devem ser acrescentadas as funções:
- LOAD
- STORE

NO ciclo de Decodificação, devem ser acrescentadas as funções:
- LOAD
- STORE
- LOADI  ok
- STOREI
- MOV  ok
- ADD/SUB/MUL/DIV/LMOD/LAND/LOR/LXOR/LNOT  ok
- INC
- CMP
- CALL
- PUSH
- POP
- RTS
- HALT
